notifications Trait¶

Used to customise build result notifications (most prominently sending error mails upon errors).

Attributes

name required? default type explanation
default no {‘on_error’: {‘triggering_policy’: ‘only_first’, ‘email’: True}} Dict the default notification cfg (more may be defined)
