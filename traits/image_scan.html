filters Attributes

name

required?

default

type

explanation

include_image_references

no

()

str

a list of regular expressions. If configured, only matching image references are processed. By default, all image references are considered.

exclude_image_references

no

()

str

a list of regular expressions. If configured, matching image references are exempted from processing. Has precedence over include_image_references. By default, no image references are excluded.

processing_mode Enumeration Values

value

explanation

upload_if_changed

upload the container images. This will not upload the images if they are already present. This will not cause images already present to be rescanned.

rescan

trigger a scan of container images. Images will be uploaded unless they are already present.

force_upload

always upload the images. This will cause all images to be rescanned.
